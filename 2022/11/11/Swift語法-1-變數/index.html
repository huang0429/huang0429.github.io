Swift語法#1-變數

發表於 2022-11-11 分類於 Swift

什麼是變數？

變數是在某一塊記憶體位址中所存放的資料、
只要建立好，就可以方便調用。

建立變數

let number: Phone = "0909-123-456"

let 是”宣告”的關鍵字
number 是變數名稱
Phone 是資料類型
0909-123-456 存在變數裡的值

“宣告”的關鍵字: let Var
let 是宣告後值無法再改變
Var 是宣告後值可以改變，但是資料類型不可以變。

練習

let name = "Jane"
let location = "Taiwan"

print("Hello", name, "from", location, separator: "-")


// 結果
Hello-Jane-from-Taiwan

separator 是在兩個變數之間的空白處做分隔

變數命名規範

  • 不能用數字開頭
  • 不能有數學符號
  • 不能有箭頭
  • 避免使用關鍵字
  • 明確的命名，如：x,y,z….

命名習慣：

  • 小駝𡶶命名法(建議)

開頭第一個單字要小寫
如： countOfApples

  • 大駝𡶶命名法

開頭第一個單字要大寫
如： CountOfApples

一個專案要維持一種寫法，看起來會比較舒服
