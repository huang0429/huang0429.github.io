新增虛擬手機

發表於 2022-07-16 分類於 XCode

新增虛擬手機型號

我的系統版本

XCode : 13.4.1

開始…

今天要寫第一份ios手機的作業

要調整虛擬手機時，發現 XCode 一個都沒有出現，所以就自己動手加一下。

不過，因為老師還沒有教，所以就上網自己找一下文章了。

點選 Add Additlonal Simulators

會出現很多選擇…

選個你想要的

然後按下最底下的 +

會出現一個視窗

然後出現意外了

就是我沒有版本可以選

所以我要另外找文章了…

不過不知道是不是我關鍵字下錯，資料蠻少的。

但是從上面看來，我的 XCode 應該沒有任何的版本，所以我找了一下可以下載版本的選項。

進入 XCode ->Preferences

會出現一個視窗

選 Compontnes：

每個選項前面都有個下載的圖示

我選了最新版本來下載

內容很大，要5G多…

我人在外面呢

幸好我有吃到飽

趁下載時間我來發想一下我的作業內容好了…

下載好了

下載好了

關閉 XCode

真正的關閉是連 Dock 上的 XCode 的圖示下面的小點點都消失了，才是真正的關閉。

再重新打開 XCode

點一下上方的圖示

就會看到手機都出現了

接著來測試一下虛擬手機可不可以啟動

可以哦!!
