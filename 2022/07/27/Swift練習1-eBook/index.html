主題：電影列表

主題為電影列表

主要有三部份：
10部必看愛情電影
10部必看好萊塢電影
必看驚悚電影

主要元件功能：

  • Label
    • 文字單行及多行輸入
  • Button
    • 跳頁
  • Texe View
    • 文字多行輸入
    • 超過 texe 輸入框時，會有卷軸。
  • Image View
    • 插入圖片
  • Scroll View
    • 可加入多張圖片
    • 利用函數，可有分頁功能。
  • Visual Effect View
    • 疊在圖片上方，可模糊圖片。
  • Table View Cell
    • 在一個cell裡面製作一個單元，然後可增加多個相同的單元。
    • 若超過螢幕，可上下卷動。
  • Navigation Controller
    • 可增加分頁。
    • 讓每個子頁面都有一個 black 按鈕。
    • 在 Navigation item 的 Title 輸入文字，會替換掉 black 字樣。
